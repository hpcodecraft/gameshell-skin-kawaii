## Installation

1. SSH into your Gameshell
2. Run `bash <(curl -s https://raw.githubusercontent.com/hpcodecraft/gameshell-skin-kawaii/master/run.sh)`
3. On your Gameshell, go to Settings->Themes, scan and select "kawaii"
